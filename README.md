# 🎓 Student Performance Predictor

A simple Machine Learning project that predicts a student's marks based on the number of study hours. This project uses **Linear Regression** to establish a relationship between study time and performance.

---

## 🚀 Features

- 📊 Predicts marks based on study hours
- 🤖 Implements Linear Regression using scikit-learn
- 💻 Clean and interactive terminal-based output
- 📚 Beginner-friendly and easy to understand

---

## 🛠️ Technologies Used

- Python 🐍
- NumPy
- Pandas
- Scikit-learn

---

## ⚙️ How It Works

The model is trained on a small dataset containing:
- Study Hours
- Corresponding Marks

It learns the relationship between these variables and predicts marks for new user input.
